## Required Fields
- Title
- Environment
- Component
- Severity
- Priority
- Preconditions
- Steps to Reproduce
- Expected Result
- Actual Result
- Evidence (screenshots, logs)
- Notes / Possible Root Cause (optional)

## Title Format
[Component] - Summary of the issue

## Steps Format
Numbered, each step must contain:
- User action
- System reaction

## Evidence Rules
- Always provide screenshot or log snippet.
- Redact sensitive data.
